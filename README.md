# VISIO
<img src="https://github.com/2IDS-A-TAMB-2026/VISIO/blob/main/imagens/imagem_logo.png" width="500">
Plataforma para identificação de sensores IOT

## Problema
- Dificuldade dos alunos em reconhecer sensores IOT.
- Dificuldade em entender aplicações reais.
- Pouca conexão com inovação e tecnologia.
- Falta de preparo para o mercado de trabalho.

 ## Objetivo
- Desenvolver uma plataforma que centralize informações sobre sensores IOT.
- Permitir que os alunos encontrem sensores de forma simples e rápida.
- Apresentar exemplos de uso dos sensores em projetos reais.
- Garantir conteúdo fácil de entender para diferentes faixa etárias.
 
 ## Público-Alvo
- Instituições de ensino.
- Cursos técnicos.
- Alunos.
   
## Funcionalidades
- Cadastro de sensor.
- Visualização detalhada.
- Conteúdo educativo.
- Interface simples.
- Atualização de conteúdo.
